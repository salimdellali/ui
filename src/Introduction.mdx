# @salimdellali/ui

- My personal, opnionated, React component library, pure CSS, zero config, fully typed.
- `npm` package coming soon.

## What is this?

This Storybook documents every component and token in `@salimdellali/ui`. It is the living reference for the library: each entry shows the component in isolation, its variants, and its props.

## What to expect

- **Tokens**: design primitives: colors, typography scale, and spacing. These are the raw CSS custom properties the components are built on.
- **Atoms**: the smallest building blocks (headings, text, etc.). Each atom maps directly to a token or a small composition of tokens.
- More categories (**Molecules, Organisms**) will appear here as the library grows.

## How to use

Use the **sidebar** on the left to browse components. Each story has:

- a **Canvas** tab to interact with the component and tweak its props via the controls panel
- a **Docs** tab for the full API reference

Use the **theme toggle** in the top toolbar to switch between light and dark mode.
