# @salimdellali/ui

My personal, opinionated React component library. Pure CSS, fully typed.

## What to expect

- **Tokens**: design primitives — colors, typography scale, and spacing
- **Atoms**: the smallest building blocks (headings, text, etc.)
- More categories (**Molecules, Organisms**) will appear as the library grows

## Browse the library

Use the **sidebar** on the left to explore components. Each story has:

- a **Canvas** tab to interact with the component and tweak its props via the controls panel
- a **Docs** tab for the full API reference

Use the **theme toggle** in the top toolbar to switch between light and dark mode.

## Install the package

```bash
npm i @salimdellali/ui
```

Then add one import to your app entry point:

```js
import '@salimdellali/ui/styles'
```
